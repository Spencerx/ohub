Introduction

Open Hardware Platform

User documentation for the open hardward design platform.

Contribute

Anyone can contribute to this documenation and any kind of contribution is welcome. Feel free to submit pull requests and issues.

Documentation structure is very simple to understand. It's basically a collection of markdown documents. We use Gitbook to generate the documentation website.

You can easily build the website locally using Gitbook CLI on you computer. First you need npm, the javascript package manager in order to install Gitbook.

Then install the gitbook CLI

$ npm install -g gitbook-cli

Note

If you have a development environment set up with vagrant already, the gitbook CLI is already installed.

Now you are ready to generate the documentation website. You can either run a server locally.

$ gitbook serve

or generate a static HTML website

$ gitbook build
